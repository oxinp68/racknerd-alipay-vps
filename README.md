## 六、付完之后：拿到 VPS 的下一步

订单付完几秒钟，邮箱就会收到一封 "Welcome" 邮件，里面带 root 密码、IP 地址、SolusVM 控制面板登录信息。之后的标准动作：

1. **SSH 登录**：终端跑 `ssh root@你的IP`，输密码进去。
2. **改 root 密码**：`passwd` 命令换一个强密码，默认密码别留。
3. **更新系统**：Ubuntu/Debian 跑 `apt update && apt upgrade -y`，CentOS 跑 `yum update -y`。
4. **装环境**：用宝塔面板、1Panel 或者直接手动装 Nginx/Docker，看你习惯。
5. **开防火墙**：`ufw allow 22`、`ufw enable`，别裸奔。

这套流程无论你买的是哪个套餐都通用，跟支付宝付款没什么关系，但很多新手卡在这一步，顺手写一下。
